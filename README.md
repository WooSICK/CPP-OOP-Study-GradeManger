# 📊 Simple Student Grade Management System (C++)

C++ 기초 문법을 익힌 후, **객체지향 프로그래밍(OOP)**의 핵심 원리를 실전 코드로 구현해 본 첫 번째 프로젝트입니다. 2시간 동안 수많은 시행착오와 버그를 해결하며 완성했습니다.

## 🚀 프로젝트 개요
- **개발 기간**: 2026년 1월 (약 2시간)
- **개발 환경**: C++, Windows Console
- **주요 기능**: 학생 추가, 전체 목록 출력, 학번 검색, 학생 정보 제거

## 🧠 핵심 학습 포인트 (What I Learned)

### 1. 객체지향의 캡슐화 (Encapsulation)
- 멤버 변수를 `private`으로 보호하고, 외부에서는 `Getter(return_number)` 함수를 통해서만 데이터에 접근하도록 설계하여 데이터의 오염을 방지했습니다.

### 2. 생성자와 메모리 관리
- 객체 배열을 생성할 때 **기본 생성자(Default Constructor)**가 왜 필요한지, 그리고 매개변수가 있는 생성자를 만들었을 때의 제약 사항을 몸소 체험하며 해결했습니다.

### 3. 배열 데이터 삭제 알고리즘 (Shifting)
- 배열 중간의 데이터를 지웠을 때 생기는 빈 공간을 메우기 위해, 뒤에 있는 데이터를 한 칸씩 앞으로 당기는 **Shifting 로직**을 직접 구현했습니다.

### 4. 사용자 인터페이스(UI) 흐름 제어
- `while`문과 `switch-case`를 조합하여 실제 서비스처럼 작동하는 메뉴 시스템을 구축하고, `system("pause")`와 `system("cls")`를 활용해 콘솔 화면을 제어했습니다.

## 🛠️ 실행 화면 예시
- 1번 메뉴: 학생 정보(이름, 학번, 성적) 입력 및 저장
- 2번 메뉴: 저장된 모든 학생 리스트 확인
- 3번 메뉴: 특정 학번 검색 및 정보 출력
- 4번 메뉴: 학번을 기준으로 데이터 삭제 및 리스트 재정렬
